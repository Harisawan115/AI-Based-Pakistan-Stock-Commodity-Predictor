# PSX Multi-Asset Forecasting Dashboard

A production-ready machine learning system for forecasting precious metals (Gold, Silver, Platinum) and USD/PKR exchange rates, powered by multiple predictive models and deployed through an interactive Streamlit dashboard.

---

## Overview

This project implements a multi-model forecasting framework designed to analyze and predict price movements of:

* Gold
* Silver
* Platinum
* USD/PKR Exchange Rate

The system integrates feature engineering, technical indicators, model training, scaling pipelines, and real-time prediction through a user-friendly web interface.

---

## Key Features

* Multi-Asset Price Forecasting
* Machine Learning Models:

  * Linear Regression
  * Random Forest
  * LSTM
* Technical Indicators:

  * Moving Averages
  * RSI
  * MACD
  * Volatility
  * Bollinger Bands
* PSX Data Connector Module
* Modular Architecture (separate training, scaling, prediction layers)
* Interactive Streamlit Dashboard
* Docker-ready structure for containerized deployment

## Installation & Local Setup

### 1. Clone the Repository

git clone <https://github.com/Harisawan115/AI-Based-Pakistan-Stock-Commodity-Predictor>
cd psx-forecast-dashboard

### 2. Create Virtual Environment (Recommended)

Windows:

python -m venv venv
venv\Scripts\activate

Mac/Linux:

python -m venv venv
source venv/bin/activate

### 3. Install Dependencies

pip install -r requirements.txt

### 4. Run the Application

streamlit run app.py

The dashboard will open in your browser at:

http://localhost:8501
